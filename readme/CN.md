# KeiOS

[English Version](../README.md)

KeiOS 是一个 Android 工具台，聚合系统参数查看、本地 MCP 服务、GitHub 版本追踪、Blue Archive 辅助与学生图鉴功能。应用使用 Compose + Miuix 构建，并提供液态玻璃风格界面、快捷状态卡、导入导出、通知提醒和缓存诊断。

## 主要功能

- Home 仪表盘集中显示 OS、MCP、GitHub、BA 状态。
- OS 工具支持系统表、Android/Java/Linux 属性、活动快捷入口、Shizuku Shell 与 Shell 卡片。
- MCP 页面支持本地服务开关、配置复制、工具总览、日志导出、前台保活和测试通知。
- GitHub 页面支持 APK 项目版本追踪、预发版策略、Release 文件下载、分享链接导入和本机应用联动。
- BA 办公室支持 AP、咖啡厅来访、竞技场刷新提醒、分服务器时区、媒体设置和学生图鉴入口。
- 学生图鉴支持目录搜索排序、媒体缓存、BGM 收藏、鉴赏媒体、媒体导出和收藏导入导出。
- 设置页提供主题、动效、液态玻璃组件、底栏特效策略、背景图、权限、缓存诊断、日志与通知兼容配置。

## 当前分发方式

- 稳定版安装包通过 [GitHub Releases](https://github.com/hosizoraru/KeiOS/releases) 发布。
- 当前稳定标签：[v1.2.3](https://github.com/hosizoraru/KeiOS/releases/tag/v1.2.3)。
- 正式版基线：`os.kei`、`arm64-v8a`、Android 15+（`minSdk 35`）。
- 运行与构建基线：`targetSdk=37`、Java 21、Kotlin/Gradle 项目工具链。
